## Comprehensive Documentation and Config Command

Comprehensive user-facing documentation covering installation, configuration, usage, and keyboard shortcuts. Previously, the project had only a one-line README, making it impossible for new users to get started without reading source code.

The README now includes a full project overview, feature highlights, quick start guide, detailed installation instructions, configuration reference, and usage examples. A complete keyboard shortcuts reference covers both dashboard navigation keys and Zellij-level shortcuts. The new `dot-agent-deck config` CLI command displays the current configuration including socket path, hook directory, and Zellij integration status—useful for verifying setup and debugging connectivity issues.

The project is now published under the MIT license.
